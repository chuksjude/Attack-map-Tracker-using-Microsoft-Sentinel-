Attack map Tracker using Microsoft Sentinel 


This project demonstrates an understanding of an attack map with real time monitoring of hacker activities using both Log Analytics Workspace and Microsoft Sentinel (SIEM). 

Objective


We simulate how logs from activities across our environment are collated by the Log Analytics Workspace and forwarded to Microsoft Sentinel (SIEM). In this lab, we would see how attacker(s) threat activity(s) use the internet as a honeypot and how logs are forwarded to a central repository in Azure and then, integrate Microsoft Sentinel to analyze real-world attack data with the geolocation where the attack is coming.
